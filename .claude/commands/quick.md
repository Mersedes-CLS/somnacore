Быстрая задача: $ARGUMENTS

Определи слой по префиксу:
- **firmware** / **fw**: только src/
- **backend** / **be**: только server/
- **miniapp** / **ui**: только server/public/

Если задача требует нескольких слоёв — скажи, не делай молча.
