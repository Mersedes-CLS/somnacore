# Частые корневые причины в SomnaCore
- I2C зависание — SDA залипает → bus recovery (bit-bang 16 SCL)
- VL53L0X timeout — RESULT_INTERRUPT_STATUS не проверяется
- distToWeightKg → NaN — пустой буфер или все outliers
- liveData stale — ESP32 не шлёт или updatedAt > 5с
- Polling leak — setInterval не очищен при смене вкладки
- Railway cold start — первый запрос timeout, ESP32 не ретраит
- Калибровка пустая — GET /calib/table на старте ESP не отработал
